ОФС

In this section you can see data about employees of company which are registered in system Quantum. Organizational functional schema (OFS) - this is a schema, where is displayed all functions in company and subordination between them. OFS lets efficiently control company due to the fact that regulations, repeated tasks, statistics and other tools of control atach not to specific people, but to the functions.

In Quantum departments, divisions, services and else named functions.

In order to build OFS, click on "Add function". This button displayed just in systems where OFS is not already built.

Employees: overall number of employees, registered in system.

Supervisors: number of employees, which set as administrator in system.

Attendance today: number of employees, which log in the system.
